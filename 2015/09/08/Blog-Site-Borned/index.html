Blog Site Borned

This is my first blog website. I use Hexo which is a very very not only Simple but also Lightweight blog framewook
to generate static resources, then depoly them to Github. In this way, I don’t have to
rent a specific remote computer to store the resources or register a Domain Name of my own. The only stuff I should
care is creating a repository named Github_Account_Name.github.io.
It’s a amazing TOOL!
I like it just because it keep a simple way to lead me to solve the problem gracefully.
